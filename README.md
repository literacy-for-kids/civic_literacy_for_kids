# Civic Literacy for Kids

![License](https://img.shields.io/badge/license-CC%20BY--NC--SA%204.0-green)

A curiosity-driven **civic literacy curriculum** that helps young learners explore how their communities, government, and democratic systems work — and how they can participate as informed, engaged citizens.

---

# About the Curriculum

This project provides an introduction to civic literacy designed for young learners (roughly ages 7-11, with adult guidance as needed).

Instead of memorizing facts about government, the curriculum helps students develop **understanding of how communities work** and how to **think critically about civic life**.

The curriculum is currently under development. The site framework is in place and lesson content will be added as it is written.

---

# Repository Structure

The curriculum is written as a documentation website built with **Docusaurus**.

Most of the educational content lives in `website/docs/`.

```
website/
  docs/
    intro.md
    license.md
```

Each lesson is written in **Markdown**, which makes it easy to edit and maintain.

The website automatically generates navigation from these files.

# Why This Curriculum Exists

Many children grow up in communities without understanding how those communities are governed, how decisions are made, or how they can have a voice.

This curriculum aims to change that.

The goal is simple:

Help kids feel like civic life is something they can understand, participate in, and help shape.
